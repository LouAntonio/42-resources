## 42 COMMON CORE

This is the Core of the 42 journey, and it defines the base to the minimun habilities (human and techniqs). It takes so long as the student needs (but don't relax), and allows the students to learn C programming, and devellop simple softwares using classig algorithms, discover the acces to the file system, learn the UNIX process management.
It also includes network architecture bases, system administration, OOP, and match more...
> [!NOTE]  
> The common Core devellop also soft skills as team work, rigor, punctuality
, responsability...

## GAMIFICATION
The 42 journey is based in gamification, and after become a CADET (oficcial student), the student becomes a rank 1, and he must devellop a serie of projects, followed by an exam, to get to the next rank and have acesses to the next projects.

## THE NORM
The 42 Normis an ammount of rules to be followed during the projects devellopment, and they serve to consolidate the good practices, and they are mandatory in all projects.
